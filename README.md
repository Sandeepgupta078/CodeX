# CodeX

Welcome to **CodeX**

## 🛠️ Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/educase-india.git
    ```
2. Navigate to the project directory:
    ```bash
    cd educase-india
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm start
    ```
